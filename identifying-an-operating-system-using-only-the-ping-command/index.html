Identifying an operating system using only the ping command.

September 21, 2021

You can identify which operating system is running simply by pinging it. All you need to do is ping localhost & read the TTL value. This should work on almost any operating system that has access to ping from a terminal. Obviously theres way better ways to find the currently running operating system, but hey this is still cool nonetheless. Simply run the ping command below in a terminal & then compare the TTL value with the table below to determine the opertaing system.
