Fixing error with updating Arch Linux keyring

May 10, 2022

If like me you happen to run Arch Linux, you may (or may not) have run into the following error while trying to update with the system.

Invalid or corrupted package PGP signature.

This problem is super easy to fix. All you need to do is run the command below to update the keyring, before then running the update once again.

sudo pacman -Sy archlinux-keyring
