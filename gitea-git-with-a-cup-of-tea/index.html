Gitea - Git with a cup of tea.

October 23, 2020

Gitea is a painless self-hosted Git service. It can run anywhere that you can compile Go, be it Linux, Bsd, Windows or macOS etc. Installing is as easy as running the provided binary for your choosen platform or by using docker or vagrant. You can also find it already packaged for a number of operating systems. The low minimal requirements means that you can run Gitea on an inexpensive Raspberry Pi. The project is completly open source and you can check it out on GitHub….
