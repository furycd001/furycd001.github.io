How to launch applications with sudo using rofi.

March 15, 2022

First off you’ll want to start by checking to see if pkexec is installed on your system. It wasn’t on my arch system, but it’s included with systemctl-policy so I just installed that to get it. Once installed I created the below bash script & saved in /usr/bin/ as ssudo. With the script created I can now launch applications with sudo using rofi….

#!/usr/bin/env bash
pkexec $1

All I have to do is open rofi & type ssudo $nameofapplication

Note that you can save the script with any name of your choice, but really ssudo kind of makes sense.
