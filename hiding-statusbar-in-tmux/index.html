Hiding statusbar in tmux.

December 27, 2021

If you are someone like me who occasionally uses tmux, but has no need for the statusbar then this post is for you. All you need to do is add the line below into your ~/.tmux.conf.

set -g status off

There was no tmux.conf by default on my system so I had to create it. You’ll probably need to do the same..
