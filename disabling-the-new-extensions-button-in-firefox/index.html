Disabling the new extensions button in firefox.

January 19, 2023

The new extensions button introduced in the lastest Firefox update is for me super annoying & pretty useless. I only have one extension installed on Firefox & have no plans to ever stop using it or remove it. Quoting mozilla from this link they said that it’s not currently possible to move or remove the button..

No, it is currently not possible to remove this button or change its location. This is by design. The extensions toolbar button is now the primary user interface and allows access to per-site permissions for MV3 extensions.

After a little research, it turns out that you can actually hide the button. You just need to switch the preference below from true to false in about:config. By default it is set to true, so just flip it to false & restart Firefox for the change to take effect.

extensions.unifiedExtensions.enabled
