Installing Arch on termux.

January 4, 2022

Ever wanted to have a portable Arch install that you can take with you whever you go :? Well with the help of an android phone, termux & this bash script you can have just that. It’s not super hard to get set up & it doesn’t take all that long either.

  • To begin you’ll want to go ahead & install termux if you haven’t already done so. Once we have termux installed we can then go ahead & install the base OS.
pkg update && pkg install bsdtar wget proot tergent tmux openssh
  • With the base OS installed you’ll now want to set up storage for termux.
termux-setup-storage
  • When you’ve finished setting up storage we can now download the script & begin setting up Arch.
wget https://raw.githubusercontent.com/SDRausty/termux-arch/master/setupTermuxArch.bash
  • When the script has finished downloading, we can then proceed to run it.
./setupTermuxArch.bash

This will download and install the base Arch system. The time this will take to finished will vary depending on your device & connection.

  • When everything has finished you’ll be dropped into a root shell for your new Arch install. We can now go ahead & change the username & password for our new install
useradd -mG root -s /bin/bash newuser
passwd newuser

Remember to change newuser to whatever you want your username to be..

  • Once you’ve changed the username & password, we can move on to making arch a little bit more useable. We’ll start by installing wget & then downloading one final script.
sudo pacman -Syyu wget && wget --no-check-certificate  'https://docs.google.com/uc?export=download&id=107Fh0l_p0ItVkUufOhP9to-OU_6KYhPW' -O fresharch.sh
  • When the script has finished downloading, we can make it executable & then run it.
chmod +x fresharch.sh && sudo ./fresharch.sh

If everything goes smoothly & you see no errors on the screen then you should have a fully functional Arch install with yay & AUR support. If you happen to come across any errors, feel free to run through all of the steps again to verify everything. This worked for me whenever I tried it, so it should work for you also….
